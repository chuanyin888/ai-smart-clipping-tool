# Windows 一键启动说明

这个版本适合直接发给别人使用。

## 用法

1. 解压整个项目目录。
2. 双击 `启动.bat`。
3. 首次运行会自动：
   - 检查 Python
   - 没有 Python 时尝试用 `winget` 自动安装 Python 3.12
   - 创建本地虚拟环境 `.venv`
   - 自动安装依赖
   - 打开图形界面
4. 在界面里输入 YouTube 地址。
5. 选择翻译方式：
   - `none`：不翻译
   - `offline`：本地离线翻译
   - `openai_compatible`：通过 OpenAI 兼容接口翻译
6. 点“开始处理”。

## 输出位置

默认输出到：

`work/<视频slug>/`

里面会有：
- `source/` 原视频和字幕
- `analysis/` 候选片段与分析文件
- `clips/` 导出的片段

## 说明

- 这个版本已经尽量做成“用户不用手动装很多东西”。
- 但第一次运行仍然需要联网准备 Python 依赖。
- `offline` 模式首次运行可能比较慢，因为会安装离线翻译相关组件。
- 如果 `yt-dlp` 遇到登录、地区限制、字幕不可用等问题，需要按实际视频情况调整。
- 如果系统里没有 `winget`，那就需要先安装 Python 3.11/3.12 再双击运行。

# FAQ

## Why does YouTube download fail with bot verification?
Some videos trigger YouTube bot verification. Update `cookies.txt` or switch to local file mode.

## Does heuristic mode consume API credits?
No. Heuristic mode is local and does not use cloud model APIs.

## Why is Start.vbs the recommended launcher?
It is the clean user-facing entry. `Start.bat` is kept only as an internal helper launcher.

## Can I upload this project to GitHub directly?
Yes, but do not upload your private `cookies.txt`, local workspace files, or personal API keys.
